## 练习 2.53 解释器在求值下面各个表达式时将打印出什么？

```eval-scheme
(list 'a 'b 'c)
```

```eval-scheme
(list (list 'george))
```

```eval-scheme
(cdr '((x1 x2) (y1 y2)))
```

```eval-scheme
(cadr '((x1 x2) (y1 y2)))
```

```eval-scheme
(pair? (car '(a short list)))
```

```eval-scheme
(memq 'red '((red shoes) (blue socks)))
```

```eval-scheme
(memq 'red '(red shoes blue socks))
```
